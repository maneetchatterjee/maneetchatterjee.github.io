## ğŸ› Troubleshooting

### Import Issues

```bash
# Add src to Python path
export PYTHONPATH="${PYTHONPATH}:$(pwd)/src"

# Or use absolute imports
from src.models.qegan_model import QEGAN
```

### PyBullet Issues

```bash
# No display available
python experiments/scripts/run_comprehensive_experiments.py --no-gui

# Permission errors
sudo apt-get install xvfb
xvfb-run -a python experiments/scripts/run_comprehensive_experiments.py
```

### Memory Issues

Reduce complexity in code:
- Decrease number of robots
- Reduce episode count
- Lower batch size
- Use CPU instead of GPU for quantum circuits

### Missing Dependencies

```bash
# Reinstall all
pip install -r requirements.txt --upgrade

# Install specific package
pip install pennylane pybullet torch-geometric
```
